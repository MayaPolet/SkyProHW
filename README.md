# SkyPro
Для домашек в курсе Python.38.0

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/MayaPolet/SkyProHW.git
```
2. Установите зависимости:
```
poetry install
```
## Использование:

1. Откройте приложение в вашем веб-браузере.
2. Созданы модули:

- generators
- masks
- processing
- widget
